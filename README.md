# e-SUS APS - Conector - Servidor

## Configurações de ambiente

- Java Open JDK 18.0.1.1
- Gradle 7.4.2

## Variáveis de ambiente

Copiar arquivo "env.example" para ".env" e preencher com as variáveis presentes no arquivo

## Execução

Linux/macOS

```sh
./gradlew bootRun
```

Windows

```sh
gradlew.bat bootRun
```
